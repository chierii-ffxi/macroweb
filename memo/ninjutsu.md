# Ninjutsu

## Utsusemi

Here, *"FastCast"* set also contains Hattori feet for Utsusemi +1.

English example:

```addon
/equipset 158 echo [FastCast]
/ma "Utsusemi: Ni" <me> <wait 1>
/recast "Utsusemi: Ni"
/equipset 164 echo [TP engaged]
/recast "Utsusemi: San" 
```

【/equipset】 7 echo 【シーフ】【ファストキャスト】
