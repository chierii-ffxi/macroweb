
## 🗺️ Translated Reference: The Minimalist JP Ranger Macro Row

The original JP site is very simple for Ranger, minimizing total macro buttons. That site actually puts RNG together with DRK to save a macro palette.
For quick reference, here is the exact mapping of his primary Ranger macro palette line:

| JP Button Label | English Translation / Game Equivalent | Notes |
| --- | --- | --- |
| **スナップ** | Snapshot | Pre-cast Snapshot gear swap. |
| **遠隔攻撃** | Ranged Attack | Hard-equips Taeon/Ikenga/Meganatada pieces for a zero-wait pull, landing instantly in Midshot Set 173. |
| **トルフラ** | Trueflight | Elemental Magic WS. |
| **ナビング** | Numbing Shot | Same set as Last Stand |
| **サベッジ** | Savage Blade | Main melee sword Weapon Skill. When using this, can use TP bonus bow and WSD +20% arrow. But plug into differnt row to avoid accidentally shooting away the arrow as I did before |
| **スカベジ** | Scavenge | `/ja "Scavenge" <me>`. Ultra-simple setup; requires no active gear modifiers. |
| **アナーキ** | Anarchy | Gun swap macro (Anarchy +2). |
| **フォマル** | Fomalhaut | Gun swap macro (Fomalhaut). |
| **ラススタ** | Last Stand | Archery/Marksmanship physical WS. |
| **サベッジ** | Savage Blade | Duplicate slot for emergency/rapid access. |
| **★移動** | ★ Movement / Warp | Dual-purpose utility slot: Equips `Shneddick Ring` for movement speed and `Warp Ring` for warp. |
| **スペジグ** | Spectral Jig | `/ja "Spectral Jig" <me>`. Instant Sneak/Invis shortcut; requires no active gear modifiers. |
