
## 🗺️ Translated Reference: Multi-Page JP Avatar Macro Palette

> ### ⚠️ Disclaimer on Completeness & Scope
> 
> 
> The original JP layout resource serves as an **excellent spatial and structural template for UI navigation**, but it is not built for absolute job completeness. The original author seems to utilize SMN primarily as an endgame utility tool rather than a specialized main job.
> Notably, this reference completely omits avatars such as **Fenrir** (*Lunar Cry*, *Eclipse Bite*), **Cait Sith** (*Mewing Lullaby* control dynamics), and **Carbuncle**. The configurations below should be used purely to understand standard layout mapping.

The original JP setup maps each primary Avatar to its own dedicated page configuration. Every Avatar profile maintains consistent positioning for basic commands while hot-swapping the specific Blood Pact Rage/Ward actions.

### 👥 Universal Command Registry (Found on every profile)

* **攻撃 (Assault):** `/pet "Assault" <t>` → Initiates avatar combat.
* **退避 (Retreat):** `/pet "Retreat" <me>` → Pulls the avatar back to your side.
* **帰還 (Release):** `/pet "Release" <me>` → Desummons the active avatar.
* **AA (Auto-Attack / Avatar Action):** Optional personal engage/TP line.
* **加護 (Avatar's Favor):** `/ja "Avatar's Favor" <me>` → Critical passive trait deployment.
* **★移動 (Movement):** Quick-swap to `Shneddick Ring` or movement speed feet.
* **FC (Fast Cast):** Pre-cast Fast Cast swap macro for non-summoning spells.
* **アポジー (Apogee):** Pre-mapped directly next to heavy Rage pacts for rapid-fire back-to-back executions.
